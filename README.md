# API Unificada de CRM e Email

Este projeto foi desenvolvido para fornecer integração entre CRMs e plataformas de email marketing. Ele foca em inclusão, escalabilidade e facilidade de uso tanto para desenvolvedores quanto para usuários não técnicos.

## Principais Funcionalidades
- Sincronização de leads entre CRMs (ex.: Salesforce, Zoho) e plataformas de email.
- Automação de campanhas de email com templates pré-aprovados.
- Relatórios e análises abrangentes.
- Documentação interativa e tratamento de erros acessível.

## Estrutura do Projeto
O projeto segue uma arquitetura modular para garantir escalabilidade e facilidade de manutenção.
